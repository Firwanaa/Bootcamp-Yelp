# Bootcamp-Yelp

YelpCamp is a website where users can create accounts and add photos about camp sites witht the functionality to post, edit and delete.
Moreover, users can comment to each others posts.
Technologies used:
HTML5

CSS3

JavaScript

Bootstrap 4

SemanticUI

DOM Manipulation

jQuery

Unix(Command Line) Commands

NodeJS
